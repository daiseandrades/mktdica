Como usar o Claude

01

Crie sua conta

Acesse claude.ai no navegador e crie uma conta gratuita ou faça login.

02

Copie o prompt

Use o botão de cópia em cada card desta página para copiar o prompt completo.

03

Personalize

Preencha seu conteúdo, tom e identidade visual. Os campos [entre colchetes] são os que você substitui.

04

Cole e envie

Cole o prompt no chat do Claude e pressione Enter. O resultado é gerado na hora.

05

Ajuste no chat

Quer alterar algo? Basta dar o comando no chat que o Claude faz as alterações.

Carrossel 100% dentro da HUB

Como criar o carrossel

1

Acesse claude.ai no navegador e crie sua conta gratuita (ou faça login)

2

Copie o prompt completo abaixo usando o botão "Copiar prompt"

3

Preencha com seu conteúdo, tom e identidade visual nos campos indicados

4

Cole tudo no chat do Claude e pressione Enter. O arquivo HTML será gerado

Como baixar as imagens

Clique em "Abrir no navegador" no arquivo gerado (use o Chrome)
Clique em "Baixar todos os slides" para baixar tudo de uma vez, ou slide por slide
Os arquivos PNG são salvos automaticamente na pasta de Downloads
Importe no Instagram na ordem dos slides e publique
Para ajustar algo, é só dar o comando no chat do Claude

O prompt para usar agora

Prompt completo

PROMPT COMPLETO PARA CARROSSEL (INSTAGRAM) PREENCHA ABAIXO: CONTEÚDO: [escreva o conteúdo completo ou assunto do carrossel] QUANTIDADE DE SLIDES: [ex: 6 slides incluindo capa e final] TOM: [ex: didático / inspiracional / vendas / informativo] IDENTIDADE VISUAL: Cores: [coloque aqui as cores da sua marca] Fontes: [coloque aqui as fontes da sua marca ou deixe a IA decidir] Handle: @[seu perfil] COMANDOS: Crie um carrossel completo para o Instagram no formato 1080x1350px considerando as informações acima. Crie um arquivo HTML para que a pessoa consiga abrir no navegador e baixar as imagens. Inclua: • Botão para baixar cada slide em PNG • Botão para baixar todos os slides de uma vez ESTRUTURA VISUAL: • Alternar entre fundo claro e fundo escuro • Headline grande (mínimo 80px) que pare o scroll • Subtítulo curto • Seta indicando swipe • Slides de conteúdo com: • Blocos com ícone + label em caps + descrição • Número decorativo grande no canto • Slide final com: • Fundo claro • Frase de impacto grande • Dois botões de CTA REGRAS DE DESIGN OBRIGATÓRIAS: • Título da capa: mínimo 80px, peso 800, legível à primeira vista • Títulos dos slides: mínimo 44px, com hierarquia clara • Alto contraste em todos os textos (nunca texto claro em fundo claro) • Cantos decorativos em todos os slides (quadrado arredondado com borda colorida cortado pela borda) • Linha de acento colorida antes de cada título • Handle (@perfil) presente no rodapé de todos os slides • Layout limpo, com bastante espaço em branco • Numeração decorativa grande e transparente nos slides de conteúdo REGRAS DE LAYOUT E DISTRIBUIÇÃO (CRÍTICO): • Todo o conteúdo deve caber 100% dentro do slide (1080x1350px), sem cortes • Proibido overflow (nenhum texto pode sair da área visível) • Conteúdo centralizado (evitar excesso no topo ou rodapé) • Espaçamento consistente entre blocos • Não sobrepor textos ou elementos • Garantir alinhamento organizado e hierarquia visual clara
